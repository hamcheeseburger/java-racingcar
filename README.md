## 문자열 덧셈 계산기 요구사항

- [x]  사용자 입력 값이 공백 혹은 null인 경우 0을 반환한다.
- [x]  입력 값에 구분자가 없는 경우 숫자를 그대로 반환한다.
- [x]  컴마 혹은 콜론이 구분자로 지정되면 구분된 숫자들의 합을 반환한다.
- [ ]  “//” 와 “\n” 문자 사이에 커스텀 구분자를 지정할 수 있다.
- [x]  음수 혹은 문자열이 입력값으로 들어온 경우 RuntimeException 예외를 발생시킨다.
